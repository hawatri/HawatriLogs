Yet Another Example Blog

This is a demonstration of how to add a new blog post to Hawatri.

Here's what you need to do to add your own blog posts:

  • Create a new HTML file in the blog folder
  • Add the blog metadata to index.json
  • Make sure the slug matches the filename
  • Refresh the page to see your new post

That's it! Your new blog post will appear in both the contents page and recent blogs.
